Project voor Mobile and Internet 2

Omschrjving

Wat wil ik doen?

Het project waaraan ik in de eerste instantie dacht is om te creëren voor het tweede semester van Mobile and Internet 2 een memory spel.

Idee

Memory is een gezelschapsspel dat een beroep doet op het kortetermijngeheugen van de spelers. Dit wil zeggen dat de spelers snel vergeten welke kaart ze omgedraaid hebben, waardoor het spel op zich moeilijk te spelen is. Dit is het spel waarover ik nu dacht, maar kan nog veranderen indien ik een beter idee heb.

Doel ?

Het doel van het spel is om twee keer dezelfde kaarten te vinden. Wanneer je de kaarten vindt, verwijderen ze van het scherm. Wanneer je het spel uit krijgt, ga je naar een hoger level, totdat je de schatkist kunt openen dankzij een code-combinatie die je door het spel heen verzamelt.

Opbouw van het spel

Voor het spel dacht ik om een thema te kiezen. Hiervoor heb ik voor een mangaserie gekozen namelijk One Piece. Voor de “background” van de kaarten, dacht ik om de doodskop te zetten die naar de piraten verwijst. Voor de andere kant van de kaarten dacht ik om de personages van de manga te kiezen. Ook nog wil ik tussendoor een wereldmap zetten zodat de speler een beeld heeft van de levels die hij al gespeeld heeft en of nog zal spelen om de code te kraken.
